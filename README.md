# Rust Practice using Book

In this repository, I practice rust using [The Rust Programming Language](https://doc.rust-lang.org/book/) book. Each directory is based on each individual chapter, and each chapter has its `notes.md` file with summary or conclusions of the chapter.

- **Chapter 1**: Installation and Introduction, Cargo, Build and Run
- **Chapter 2**: Project, Variables, IO, Random, String functions (trim, parse), Annotated types, Pattern matching and comparison, Ordering less greater equal, loop, custom Result handling
- **Chapter 3**: Variables, mutability, const, shadowing
